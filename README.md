# Web3 Token Transfer CLI

This project lets you send tokens.
It works on Ethereum testnet.
You can learn how ERC20 transfers work.

This project is made for beginners.
All code is simple and clear.

## Features
- Send ERC20 tokens
- Check token balance
- Command line interface
- Simple config file

## Requirements
- Node.js installed
- Testnet wallet
- ERC20 token contract
- RPC provider account

## Setup
1. Clone this repo
2. Open folder in terminal
3. Run: npm install
4. Create a file called .env
5. Add private key and RPC URL
6. Run: node transfer.js

## Example
node transfer.js 0xReceiverAddress 10
The token will be sent.

## Learning Goals
- Learn token standards
- Learn blockchain transactions
- Learn wallet signing
